How to Add Your Own Tools

  1. Create a new repository for each tool on GitHub
  2. Add a release with your ZIP file in the repository
  3. Create a redirect file in this repository (like ritzycmd.html)
  4. Update this page with your tool's name, description, and download link
  5. Update the _redirects file to map your clean URL (/toolname) to the redirect file
